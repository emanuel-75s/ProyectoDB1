ProyectoDB - Sistema de Gestión de Inventario y Ventas

Sistema de escritorio desarrollado en Java (Swing + Maven) para la gestión de inventario y ventas de una mueblería, con arquitectura MVC + DAO + Factory y soporte para dos motores de base de datos diferentes (MySQL y Oracle), intercambiables sin afectar la lógica del programa.

Funcionalidades


Login de usuarios con validación de credenciales
Gestión de inventario: alta, edición y consulta de muebles (código, nombre, categoría, material, color, precio, stock, dimensiones)
Control de stock en tiempo real
Registro de ventas: cliente, teléfono, cantidad, precio total, fecha y tipo de pago
Reportes avanzados con generación de documentos en PDF
